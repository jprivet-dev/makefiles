# Makefiles

## Presentation

Collection of useful makefiles and commands, with Docker, Symfony, PHP, Composer, Angular, Node, ...

## Installation

```shell
git clone git@github.com:jprivet-dev/makefiles.git
cd makefiles
```

## Cheat sheet

### Structure

```
# Comments are preceded by the hash symbol
target: dependencies
    command 1
    command 2
    ...
    command n
```
